🚀 Project Spotlight: AAI Tender Database Management & Insights (2025–26)
I’m excited to share my recent project where I worked on building a comprehensive tender database management system for AAI, leveraging modern data visualization and analytics tools.

🔑 Key Highlights of the Project:
- Visualization Tools: Power BI dashboards to present vendor insights in an interactive and intuitive way.
- ETL & Data Handling: Excel, Pivot Tables, and Power Query for efficient data transformation and cleaning.
- Database Management: Writing SQL queries on Oracle Database and integrating with SAP S/4HANA Cloud for robust backend support.

- Insights Delivered:
- Total vendors engaged in FY 2025–26
- Tender categories: Services / Works
- Tender types: Open / Limited / NA
- Bid opening & publishing timelines
- Revenue generation segmented by regions
- A detailed table of Top 5 tenders
- Provided Top 5 projects by region with total expenditure
- Achieved 38% cost savings in tender evaluation through optimized insights

📊 The outcome was a data-driven view of tender activities, enabling stakeholders to track performance, identify trends, and make informed decisions with clarity and confidence.
This project reinforced the power of combining ETL, SQL, and visualization tools to transform raw data into actionable insights.
💡 Looking forward to applying these learnings to future projects and exploring deeper integrations of cloud-based analytics with enterprise systems.
